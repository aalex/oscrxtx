Pure Data objects for sending and receiving OSC messages

This a set of objects for Pure Data for sending and receiving Open Sound
Control messages. It uses liblo. It supports UDP and TCP, as well as
multicast UDP.

Dependencies
------------
Requires liblo OSC library: http://liblo.sourceforge.net/
Requires pkg-config as well.


More info
---------
Some examples and help patches are provided with this package. Please open
the patches with Pure Data.
